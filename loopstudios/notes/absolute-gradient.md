# 绝对定位与渐变详解

## 源代码

```html
<!-- Item Gradient -->
<div
  class="absolute top-0 bottom-0 right-0 left-0 bg-gradient-to-b from-transparent to-gray-900 group-hover:from-gray-50 group-hover:to-white group-hover:opacity-70"
></div>
```

## 1. 绝对定位 - 占满父元素

```css
absolute top-0 bottom-0 right-0 left-0
```

| 类名 | 作用 |
|------|------|
| `absolute` | 绝对定位 |
| `top-0` | 距顶部 0 |
| `bottom-0` | 距底部 0 |
| `right-0` | 距右边 0 |
| `left-0` | 距左边 0 |

效果：**完全覆盖父元素**，相当于 `inset: 0` 或 `width: 100%; height: 100%`

```
┌────────────────────────────────┐
│  父元素 (relative)             │
│  ┌──────────────────────────┐ │
│  │ 绝对定位元素 (top-0等)  │← 覆盖整个父元素
│  │ 覆盖在整个图片上方        │ │
│  └──────────────────────────┘ │
└────────────────────────────────┘
```

### inset 简写

Tailwind 3+ 提供 `inset-0` 可以同时设置 4 个方向：

```html
<div class="absolute inset-0"></div>
```

## 2. 渐变效果

```css
bg-gradient-to-b from-transparent to-gray-900
```

| 类名 | 作用 |
|------|------|
| `bg-gradient-to-b` | 背景渐变方向：从上到下 |
| `from-transparent` | 起始颜色：透明 |
| `to-gray-900` | 结束颜色：深灰色 |

### 渐变方向

| 类名 | 方向 |
|------|------|
| `bg-gradient-to-t` | 上 |
| `bg-gradient-to-tr` | 右下到右上 |
| `bg-gradient-to-r` | 右 |
| `bg-gradient-to-br` | 右下到左下 |
| `bg-gradient-to-b` | 下 |
| `bg-gradient-to-bl` | 左下到右下 |
| `bg-gradient-to-l` | 左 |
| `bg-gradient-to-tl` | 左上到右上 |

### 颜色变化

| 类名 | 作用 |
|------|------|
| `from-xxx` | 起始颜色 |
| `via-xxx` | 中间颜色（可选） |
| `to-xxx` | 结束颜色 |

### 本例效果

```
┌────────────────────────┐
│ 透明 (transparent)     │
│        ↓               │
│        ↓ 渐变          │
│        ↓               │
│  深灰 (gray-900)       │
└────────────────────────┘
```

## 3. 悬停效果

```css
group-hover:from-gray-50 group-hover:to-white group-hover:opacity-70
```

| 类名 | 作用 |
|------|------|
| `group-hover:from-gray-50` | 悬停时起始变浅灰 |
| `group-hover:to-white` | 悬停时结束变白色 |
| `group-hover:opacity-70` | 悬停时整体透明度 70% |

### 效果对比

```

│ ████████████████ │        │ ░░░░░░░░░░░░░░░░░ │

  深灰渐变覆盖                  白色渐变+半透明
```

## 4. 层级关系

```
┌─────────────────────────────────┐
│  父元素 (relative)              │
│  ┌───────────────────────────┐  │
│  │ 图片 (z-0)                │  │
│  ├───────────────────────────┤  │
│  │ 渐变遮罩 (absolute, z-10) │← 覆盖在图片上
│  ├───────────────────────────┤  │
│  │ 文字 (absolute, z-20)     │← 最上层
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

## 5. 实际应用场景

- **图片遮罩**：让文字在图片上更清晰
- **悬停效果**：增强交互反馈
- **渐变叠加**：创建现代感 UI
- **按钮悬停**：背景渐变变化

## 6. 注意事项

1. 父元素必须设置 `relative`，否则绝对定位会相对于更远的祖先
2. 渐变默认是透明渐变到目标色
3. `group-hover` 需要父元素有 `group` 类
